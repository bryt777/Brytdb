# Telegram Art Gallery

A modern art gallery website where all media is stored in Telegram.

## Architecture
• GitHub – code only (one time)
• Vercel – website & serverless API
• Telegram – media storage & CMS

## How it works
Upload artwork to a private Telegram channel.
Captions become titles.
Hashtags become filters.

No media files are stored in this repository.

## Features
• Art gallery grid
• Fullscreen viewer
• Zoom & download
• Tag-based filters
• Search by title or tag
• Mobile friendly
• Free forever
